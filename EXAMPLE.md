# Example Markdown Document

## Introduction

This is an **example document** to test the *Markdown Doctor* add-in for Microsoft Word.

## Features

The add-in supports the following markdown features:

### Text Formatting

- **Bold text** using `**text**` or `__text__`
- *Italic text* using `*text*` or `_text_`
- ***Bold and italic*** using `***text***`
- `Inline code` using backticks

### Headers

You can create headers from H1 to H6:

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

### Lists

#### Unordered Lists

- First item
- Second item
- Third item

Or using asterisks:

* Item A
* Item B
* Item C

#### Ordered Lists

1. First step
2. Second step
3. Third step

### Blockquotes

> This is a blockquote. It can contain multiple lines
> and will be formatted with indentation and styling.

### Links

You can create [links to websites](https://github.com/amyxdclark/MarkdownDoctorWordAddon) easily.

### Code

For inline code, use `backticks`. For code blocks:

```
function hello() {
  console.log("Hello, World!");
}
```

## How to Use

1. Highlight any markdown text in your Word document
2. Click the "Convert Markdown" button in the task pane
3. Watch your markdown transform into beautiful Word formatting!

## Tips

- **Start simple**: Try converting small sections first
- **Preview**: Check the result before converting large documents
- **Experiment**: Try different markdown syntax combinations

## Conclusion

The ***Markdown Doctor*** add-in makes it easy to bring markdown formatting into Word!

Visit the [project on GitHub](https://github.com/amyxdclark/MarkdownDoctorWordAddon) for more information.
